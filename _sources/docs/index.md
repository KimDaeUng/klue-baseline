---

## **KLUE 로 모델 평가하기**

---

**KLUE 로 모델 평가하기** 는 

1. 한국어 AI 모델의 평가 지표 [KLUE 벤치마크](https://klue-benchmark.com/) 를 경험하고,
2. 총 8개의 task 에 대한 baseline 을 제작함으로써,
3. 한국어 자연어처리의 최신 기술을 적용하고자 합니다.
